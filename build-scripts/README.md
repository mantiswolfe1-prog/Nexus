# Build & Development Scripts

This folder contains utility scripts for development and deployment.

## Scripts Overview

### Fix Scripts (fix-*.sh)
Various scripts that fix imports, components, and project structure issues.

### Push Scripts (push-*.sh)
Scripts that implement fixes and push changes to GitHub.

### Other Scripts
- `cleanup-old-files.sh` - Clean up legacy files
- `deploy-prebuilt.sh` - Deploy pre-built version
- `merge-to-main.sh` - Merge changes to main branch
- `organize-and-run.sh` - Organize files and start development

## Usage

Most scripts are part of the development workflow and have already been executed. They're archived here for reference.

If you need to use any of these scripts, run them from the project root:
```bash
./build-scripts/script-name.sh
```
